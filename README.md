# Train Management & Search System

A Spring Boot backend application inspired by real-world railway systems, designed to manage trains, stations, and schedules with efficient search functionality.

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## Features
- Manage trains, stations, and train schedules
- Search trains based on source and destination stations
- RESTful APIs with clean layered architecture
- Proper separation of Controller, Service, and Repository layers

## Architecture
Controller -> Service -> Repository -> Database

## API Overview
- Add and manage trains
- Add and manage stations
- Schedule trains between stations
- Search trains between source and destination

## Project Objective
This project demonstrates backend system design, real-world data modeling, and REST API development using Spring Boot. It focuses on writing clean, maintainable code and implementing practical business logic similar to real railway systems.
